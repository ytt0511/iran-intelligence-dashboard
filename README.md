# 伊朗情报看板 (Iran Intelligence Dashboard)

一个专业的伊朗地缘政治风险监测网页看板，用于监控伊朗相关风险及金融市场联动情况。

## 项目概述

整体风格保持专业、情报化、深色科技风，采用四板块结构展示关键情报数据。

## 四板块结构

### 第一板块：高频实时事件区（Polymarket + 航运态势）
- Polymarket中与伊朗冲突、美国军事行动、霍尔木兹海峡、能源供应等相关预测市场数据
- 关键概率变化曲线、过去1小时概率涨跌幅
- 关键航运图：波斯湾、霍尔木兹海峡、阿曼湾的船舶密度变化
- 更新频率：每小时
- 布局：左侧概率卡片，右侧航运热力图

### 第二板块：伊朗实时新闻小时报
- 伊朗国内政治军事动态
- 美国对伊朗相关表态
- 以色列、中东相关联动
- 油气设施、军事部署、制裁、袭击等关键词事件
- 展示：顶部本小时摘要 + 下方新闻时间线
- 更新频率：每小时

### 第三板块：冲击事件概率模型区
- 短期军事冲突概率
- 油价暴涨概率
- 航运中断概率
- 制裁升级概率
- 区域扩散概率
- 展示：概率仪表盘 + 趋势曲线 + 关键触发因子

### 第四板块：高频敏感资产区
- 国际油价（WTI、Brent）
- 黄金现货
- 美元指数
- 上海航运指数
- 相关期货主力合约
- 更新频率：5-15分钟
- 展示：价格数字 + 涨跌幅 + 迷你K线图

## 技术栈

- **框架**: Next.js 14.2.5 + React 18
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图表**: Recharts
- **构建输出**: 静态导出 (Static Export)

## 项目结构

```
app/
├── api/                    # API 路由 (静态导出时不使用)
│   ├── assets/
│   ├── news/
│   ├── polymarket/
│   ├── probability/
│   └── shipping/
├── components/             # React 组件
│   ├── AssetsPanel.tsx     # 敏感资产面板
│   ├── NewsPanel.tsx       # 新闻面板
│   ├── PolymarketPanel.tsx # Polymarket面板
│   ├── ProbabilityPanel.tsx # 概率模型面板
│   └── ShippingPanel.tsx   # 航运态势面板
├── lib/
│   └── mockData.ts         # 模拟数据生成器
├── types/
│   └── index.ts            # TypeScript 类型定义
├── globals.css             # 全局样式
├── layout.tsx              # 根布局
└── page.tsx                # 主页面
```

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 构建输出在 dist/ 目录
```

## 部署

项目配置为静态导出，可以将 `dist/` 目录部署到任何静态托管服务：

- Vercel
- Netlify
- GitHub Pages
- AWS S3 + CloudFront
- 阿里云 OSS
- 腾讯云 COS

## 自动刷新机制

- **新闻/事件/概率模型**: 每小时自动刷新
- **敏感资产**: 每5分钟自动刷新
- **手动刷新**: 点击顶部导航栏的刷新按钮

## 响应式设计

- 桌面端：四板块并排或网格布局
- 平板端：自适应双列布局
- 移动端：单列堆叠布局

## 特色功能

1. **深色科技风主题**: 专业的情报分析界面风格
2. **实时数据模拟**: 模拟真实的数据更新场景
3. **交互式图表**: 使用 Recharts 实现的数据可视化
4. **导航筛选**: 可按板块筛选显示内容
5. **风险等级指示**: 颜色编码的风险等级系统
